# musik
Alat untuk download musik, lirik, chord dan mengolahnya secara offline
# Cara Menggunakan
* pkg update && pkg upgrade
* pkg install python
* pkg install mpv
* pip install -r requirements.txt
* git clone https://github.com/karjok/musik
* cd musik
* python musik.py
